# Financial Q&A App

A web application that processes financial documents (PDF and Excel) and provides an interactive question-answering system using local Small Language Models (SLMs) via Ollama.

## Features
- Upload PDF and Excel financial documents
- Interactive Q&A with local SLM
- Streamlit web interface

## Setup
1. Clone the repo: `git clone https://github.com/<username>/financial-qa-app.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
